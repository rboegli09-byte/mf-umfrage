QR-Code für die Umfrage

MF 6S.180 Dyna-VT – Kundenbefragung

GitHub Pages URL eingeben

Geben Sie die URL oben ein und klicken Sie «Generieren».

So funktioniert es

  1. Laden Sie das Projekt zuerst auf GitHub Pages hoch (siehe README.md).
  2. Ihre GitHub-Pages-URL lautet:
    https://IHR-BENUTZERNAME.github.io/mf-survey/
  3. Geben Sie die URL oben ein und klicken Sie «Generieren».
  4. Laden Sie den QR-Code herunter oder drucken Sie ihn direkt aus.
  5. Platzieren Sie den QR-Code gut sichtbar am Traktor oder im Ausstellungsbereich.
